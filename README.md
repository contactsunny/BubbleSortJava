# Bubble Sort

This is a simple Java program to demonstrate how to perform bubble sort on a 
given array.

## Input

```shell script
Original Array: 
 29 10 14 37 13 2 100 
-------------------------
```

## Output

```shell script
Sorted Array in Ascending order: 
2 10 13 14 29 37 100 
-------------------------
Sorted Array in Descending order: 
100 37 29 14 13 10 2 
-------------------------
```
